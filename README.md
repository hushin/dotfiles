# Dotfiles

My dotfiles, managed with https://chezmoi.io

## Installation

<!-- TODO あとでbranchを消す -->
```
chezmoi init hushin --branch chezmoi
```

gitのユーザー設定は、`$HOME/.gitconfig.local` に書いて下さい。

```
# .gitconfig.local の例
[user]
  name  = hushin
  email = ngtv.hsn@gmail.com
```
